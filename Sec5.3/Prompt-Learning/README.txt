Installations:

We first need to install dassl and related libraries by following instructions given in https://github.com/CHENGY12/PLOT/tree/main/plot-coop

Dataset:

Create fnested olders DATA/eurosat/.
Download the dataset from http://madm.dfki.de/files/sentinel/EuroSAT.zip and extract it to $DATA/eurosat/.
Download split_zhou_EuroSAT.json from https://drive.google.com/file/d/1Ip7yaCWFi0eaOFUGga0lUdVi_DDQth1o/view.


Then please navigate to prompt-coop/scripts and run 
```bash main_<SHOT>.sh 4```.
